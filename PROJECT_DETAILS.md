 Certificate Verification System (Blockchain)

📌 Description

This project is a blockchain-based certificate verification system that ensures authenticity using smart contracts and MetaMask. It prevents forgery by storing certificate hashes on the blockchain.

 ⚙️ Installation

1. Clone the repository:
   git clone <your-repo-link>

2. Navigate to project folder:
   cd certificate-verification-system

3. Install dependencies:
   npm install

4. Run the project:
   npm run dev

5. Open in browser:
   http://localhost:3000

🔗 Requirements

* Node.js installed
* MetaMask extension installed in browser

 🚀 Features

* Issue certificates securely
* Verify certificates using blockchain
* Detect fake certificates
* Tamper-proof and decentralized system

 🧠 How it works

* Certificate data is converted into a hash
* Hash is stored on blockchain using smart contracts
* During verification, hashes are compared
* Matching hash → Genuine | Non-matching → Fake
